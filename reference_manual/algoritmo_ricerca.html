HOLIS - ALGORITMO DI RICERCA

Si prende il testo scritto dall'utente nel form di ricerca e si costruisce un array di parole usando il carattere spazio come separatore.

Si chiede al categorizzatore EUROVOC di categorizzare l'intera stringa di ricerca, usando la seguente url: http://serendipity.lynxlab.com:81/services/%lt;password>/search/HOLIS/, questa chiamata tornerà un array di ID eurovoc ed uno di sinonimi associati alla stringa passata.
Nel caso in cui il categorizzatore non dovesse trovare sinonimi, verrano chiesti a MULTIWORDNET alla seguente URL: http://serendipity.lynxlab.com:81/services/%lt;password>/taxonomy/MWNL/synonyms/

A questo punto, è stato fatto tutto il possibile per avere un array di ID eurovoc ed uno di sinonimi o comunque di termini associati alla stringa di ricerca scritta dall'utente.
Nota: Da questo punto in poi la stringa dell'utente non è più utilizzata e viene sostituita con i termini tornati dal categorizzatore o dalla ricerca dei sinonimi. (D'ora in poi chiamati "Termini di ricerca").

Se ancora l'array degli ID eurovoc dovesse essere vuoto, si cercano degli id che corrispondano ai termini di ricerca.

Il risultato della ricerca è formato dall'unione dell' insieme degli asset associati agli ID trovati con quello degli asset trovati mendiante ricerca full text sul testo dell'asset stesso usando i termini di ricerca.
